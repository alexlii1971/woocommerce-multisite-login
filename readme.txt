在WordPress多站点环境中与WooCommerce集成，支持手机号、短信、微信登录等功能。

== Description ==

本插件为WordPress多站点环境提供与WooCommerce的深度集成，支持以下功能：
- 手机号短信验证注册与登录
- 微信扫码登录
- 多站点用户同步
- GDPR合规支持
- 模块化管理与调试日志

== Installation ==

1. 上传插件文件到 `/wp-content/plugins/` 目录，或通过WordPress后台直接安装。
2. 在插件页面激活 "WooCommerce Multisite Login"。
3. 进入WooCommerce设置页面，配置短信和微信登录参数。
